    Como é uma estrutura básica de um HTML?

    A estrutura básica é composta por tags, que são partes da estrutura que iniciam com sinal de < e > com uma palavra dentro deles, fechando com a mesma estrutura com adicional de uma / na frente da palavra, pois as tags precisam ser abertas e fechadas porém há exceções de tags que não precisam ser fechadas como a /input, pois elas já vem com a barra no final a fechando. A tag inicial é a html, e tudo que fica dentro dela é considerada HTML, dentro da tag html há ainda as tags head (cabeçalho) e body (corpo).

    Entendendo a diferença entre listas ordenadas e não ordenadas

    As listas ordenadas ficam em ordem númericas, elas seguem restritamente a ordem que foi colocada dentro do html, serve para quando precisa enumerar uma ordem. As listas não ordenadas elas servem como marcadores, pois como não seguem uma ordem, vem apenas em formato de lista sem numeração ou algo similar, é bom para lista de compras.
